Create a new post file from a title, with front matter filled in and a slug
derived from the title.

```
blogin new <title> [--src=<dir>] [--section=<name>]
```

## Arguments

- `<title>` is the post title. The filename slug comes from it.

## Options

- `--src=<dir>` is the content directory. It defaults to `content`.
- `--section=<name>` is the section to place the post in. It defaults to the
  `home-section` from `blogin.json`.
- `--verbose` and `--quiet` set the logging level.

## Example

```
blogin new "My First Post" --section=posts
```
