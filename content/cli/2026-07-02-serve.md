Build the site and serve it locally, rebuilding when content, layouts, static
assets, or the configuration change.

```
blogin serve [--src=<dir>] [--port=<n>]
```

## Options

`--src=<dir>`
: the content directory. It defaults to `content`.
`--port=<n>`
: the port to listen on. It defaults to `3000`.
`--verbose`
: prints extra detail.

## Example

```
blogin serve --port=4000
```
