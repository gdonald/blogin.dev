Build the site and serve it locally, rebuilding when content, layouts, static
assets, or the configuration change.

```
blogin serve [--src=<dir>] [--port=<n>]
```

## Options

- `--src=<dir>` is the content directory. It defaults to `content`.
- `--port=<n>` is the port to listen on. It defaults to `3000`.
- `--verbose` and `--quiet` set the logging level.

## Example

```
blogin serve --port=4000
```
